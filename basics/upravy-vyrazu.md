# Úpravy výrazů

## Vlastnosti

### Kumulativita

Nezáleží na pořadí operací

- sčítání `a + b = b + a`
- násobení `a * b = b * a`

### Asociativita

Do výrazu mohu libovolně vkládat závorky

- sčítání `a + b + c = (a + b) + c = a + (b + c)`
- násobení `a * b * c = (a * b) * c = a * (b * c)`

### Distributivita

- roznásobování `a * (b + c) = a * b + a * c`
